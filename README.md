# COGS-189-Group-Project-WI26
COGS 189: Brain Computer Interfaces Final Project

## Hello World!

## Clone this repo
```bash
git clone https://github.com/jonah-davis/COGS-189-Group-Project-WI26.git
```

## Activate Virtual Environment
```bash
python -m venv .venv::
```

#### Powershell
```bash
.\.venv\Scripts\activate
```

#### Bash
```bash
source .venv/bin/activate
```

## Install Dependencies
```bash
pip install -r requirements.txt
```
